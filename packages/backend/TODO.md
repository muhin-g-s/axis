Мульти-тенантной системы управления задачами на Node.js, функционально аналогичной Jira/YouTrack. Ключевые бизнес-сущности: users (пользователи), workspaces (рабочие пространства), workspace_users (участники пространств), projects (проекты), issues (задачи), statuses (статусы), priorities (приоритеты), labels (метки), issue_labels (связь меток с задачами). напиши мне пример варианта использования где потребуется вызвать несколько модулей для осуществления общей задачи
